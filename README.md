# Twitter Clone (React + Vite)

A modern **Twitter (X) homepage clone** built using **React.js** and **Vite**.  
This project focuses on replicating the UI and basic structure of Twitter’s landing page.

---

## Features

- Twitter homepage layout
- Sidebar navigation
- Feed section (posts/tweets UI)
- Responsive design (basic)
- Fast development with Vite
- Reusable React components

---

## Tech Stack

- **React.js** – UI library
- **Vite** – Fast build tool
- **CSS** – Styling
- **JavaScript (ES6+)**
